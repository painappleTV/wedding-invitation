# 人ごとの文言の出し分け

招待コードごとに、表示する文言を変えることができます。

## 設定方法

### 1. ゲストデータに `customText` を追加

DynamoDB のゲストデータ、またはモックデータ（[src/api/mock.ts](src/api/mock.ts)）に `customText` を追加します。

```json
{
  "inviteCode": "xyz789",
  "name": "佐藤花子",
  "message": "お二人の門出を心よりお祝い申し上げます。",
  "customText": {
    "headerTitle": "披露宴のご案内",
    "sectionTitle": "挙式・披露宴のご案内",
    "venueName": "〇〇ホテル",
    "venueAddress": "東京都〇〇区...",
    "date": "2026年3月15日（日）",
    "time": "受付 11:30 / 挙式 12:00",
    "footerText": "田中 太郎 & 山田 花子"
  }
}
```

### 2. 変更できる項目

| 項目 | 説明 | デフォルト |
|------|------|------------|
| headerTitle | ページ上部のタイトル | 結婚式のご案内 |
| sectionTitle | 挙式・披露宴セクションの見出し | 挙式・披露宴のご案内 |
| venueName | 式場名 | config.ts の venueName |
| venueAddress | 式場住所 | config.ts の venueAddress |
| date | 日付 | config.ts の date |
| time | 時間 | config.ts の time |
| footerText | フッターの名前 | config.ts の coupleNames |

**指定した項目だけ**が上書きされます。未指定の項目は config のデフォルトが使われます。

### 3. モックで試す

開発時は [src/api/mock.ts](src/api/mock.ts) を編集してください。

- `/invite/abc123` … デフォルトの文言
- `/invite/xyz789` … customText あり（headerTitle, footerText を変更）

### 4. 本番（DynamoDB）で設定する

AWS コンソールの DynamoDB で該当ゲストの `customText` 属性を追加するか、seed スクリプトの `GuestInput` に `customText` を含めて投入してください。
